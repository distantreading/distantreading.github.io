How to read this graphic

The mosaic display shows how many texts satisfy each combination of three balance criteria.

  • The length criterion is represented by three columns running left to right
  • The date criterion is represented by four rows running top to bottom
  • Each cell for a given combination of date and length is divided horizontally in two for the gender criterion
  • The blue part contains the count for male-authored texts, the pink part that for female-authored texts
  • If the count for any combination is zero, the corresponding half-cell is omitted.

For example, if all texts are in the same date slot, there will be only one row; if all texts are of the same length, there will be only one column.

The width and height of each cell are drawn in proportion to the number of texts. For example, if there are twice as many texts from T4 as from T3, the T4 row will be twice as tall as the T3 row.
