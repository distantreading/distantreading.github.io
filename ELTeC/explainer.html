How to read this graphic

The mosaic display shows how many texts satisfy each possible combination of three balance criteria.

  • The length criterion is represented by the three columns running left to right
  • The date criterion is represented by the four rows running top to bottom
  • The gender criterion is represented by the two colours
Each cell contains a count of the number of texts satisfying the length and date criteria and is coloured to indicate the gender concerned: pink for male, blue for female.

If no text is found for a given combination, the row or column concerned is not divided. For example, if all texts are in the same date slot, there will be only one row; if all texts are of the same length, there will be only one column.

The width and height of each cell are drawn in proportion to the number of texts. For example, if there are twice as many texts from T4 as from T3, the T4 row will be twice as tall as the T3 row.
